# 🤝 Como contribuir com o M.E.R.U.E.M

Primeiramente, obrigado por dedicar seu tempo para contribuir! O M.E.R.U.E.M é um projeto de código aberto construído para descomplicar as finanças pessoais, e nós adoramos receber ajuda da comunidade.

## 🧠 Nossa Filosofia
A "assinatura" do M.E.R.U.E.M é a **Simplicidade e Fluidez**. 
* Evitamos excesso de informações na tela.
* Priorizamos animações suaves (microinterações) e Glassmorphism.
* O código deve ser tão limpo quanto a interface.

## 🛠️ Como começar

1. Faça um **Fork** deste repositório.
